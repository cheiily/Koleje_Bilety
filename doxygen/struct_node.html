Detailed Description

A structure uniquely representing a node/station and its relations to other nodes within a single line. Multiple lines can and should have different objects referring to the same physical station. In order to simplify interaction with such nodes it is advised that the name variable should be set to the represented physical station's name or something similarly representative, rather than a unique identifier.
