Basic ctor initializing all members. Clears the passed route set of nullptrs and checks its size.
The passed route must have at least 2 non-null members, otherwise an exception is thrown.
By default open is set to true and renovated to false.

Exceptions
EmptyStringException- if lineId is empty
CollectionMismatchException- if size of route after clearing is less than 2
NullPointerException- if assignedUnit is a nullptr

◆ setRenovatedSegment()

void Line::setRenovatedSegment ( std::vector< std::string >  renovatedSegment)

Sets the renovated segment variable. If the passed set is empty, the internal renovated segment set is cleared. Otherwise, out of the passed set, only nodes that are already present in the line's route and are not null are appended to the renovated segment set. If the passed set is not empty but only contains nullptr's the internal renovated segment set is not changed. Make sure to clear the segment every time a renovation is finished, otherwise it may retain unwanted nodes.

Parameters
renovatedSegment- vector of nodes' names in std::strings
