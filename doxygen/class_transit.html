Basic ctor initializing all members. Clears the passed route set of nullptrs and checks its size.
The passed route must have at least 2 non-null members, otherwise an exception is thrown.

Exceptions
EmptyStringException- if transitId is empty
CollectionMismatchException- if size of route after clearing is less than 2
NullPointerException- if assignedUnit is a nullptr
